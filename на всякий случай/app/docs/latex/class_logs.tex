\doxysubsection{Detailed Description}
Сервис логирования сообщений в файл. 

Записывает сообщения с меткой времени и уровнем (INF, WARN, ERR). Если основной файл не может быть открыт, использует резервный путь в директории App\+Local\+Data\+Location.
