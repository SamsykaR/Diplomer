\doxysubsection{Detailed Description}
Клиент для удалённого взаимодействия с сервером приложения. 

Устанавливает TCP‑соединение, обменивается JSON‑сообщениями. Дублирует интерфейс \doxylink{class_d_b_worker}{DBWorker} для прозрачной работы из QML независимо от режима (локальный или клиент‑сервер). При потере связи генерирует сигнал error\+Occurred.
