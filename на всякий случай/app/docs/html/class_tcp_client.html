Подробное описание

Клиент для удалённого взаимодействия с сервером приложения.

Устанавливает TCP‑соединение, обменивается JSON‑сообщениями. Дублирует интерфейс DBWorker для прозрачной работы из QML независимо от режима (локальный или клиент‑сервер). При потере связи генерирует сигнал errorOccurred.
